Cashslide - Protocol Spec

A on-chain encrypted messaging application

[timestamp] is used to let users who were not in the chat at the time of the transaction being sent out so that messages do not appear out of order. If the timestamp does not correspond with the block it was confirmed included in, it will be hidden from new users.


1-on-1 Private Chats (0x00-0x0F)

0x01 - Send/Accept friend request (share pubkeys) - [preifx] [0x01]

0x04 - Encrypted Private Message - [preifx] [0x04] [timestamp] [message]

Public Group Chats (0x10-0x1F)

0x10 - Set Plaintext Group Title - [prefix] [0x10] [title]

0x11 - Add Moderator - [prefix] [0x12] [address]

0x12 - Restrict User Posting - [prefix] [0x13] [address]

0x13 - Unrestrict User Posting - [prefix] [0x14] [address]

0x14 - Plaintext Group Message - [preifx] [0x15] [timestamp] [message]

Tier 1 Private Group Chats (0x20-0x2F)

0x20 - Set Encrypted Group Title - [prefix] [0x10] [title]

0x21 - Add Moderator - [prefix] [0x12] [address]

0x22 - Restrict User Posting - [prefix] [0x13] [address]

0x23 - Unrestrict User Posting - [prefix] [0x14] [address]

0x24 - Encrypted Group Message - [preifx] [0x15] [timestamp] [message]
